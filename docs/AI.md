## AI Assistance

### Tool Used
**Claude** (Anthropic) 

### How it was used

#### A-AiAssistance
Claude was used to add comprehensive Javadoc comments to methods and classes that did not already have Javadoc comments.

Claude was also used to improve existing Javadoc comments to be more specific and follow the course standards. 

#### A-BetterGui
Claude was used to refine the GUI by 
1. Cropping profile pictures as a circle
2. Highlighting errors, text will be shown in red if an error is encountered
